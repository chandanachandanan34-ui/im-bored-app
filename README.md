#  Im-Bored-App

An interactive **Node.js + Express.js** web application that fetches data from the **Bored API** to suggest random and filtered activities.

## 🚀 Features
- Get random activity suggestions
- Filter activities by type and participants
- Dynamic UI using EJS templating
- API integration using Axios

## 🛠️ Tech Stack
- Node.js
- Express.js
- EJS
- Axios
- Bored API
